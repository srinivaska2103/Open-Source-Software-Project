Pre-flight Checklist

Ensure your environment meets these minimum requirements before initiating the installation process.

Version 2.4.0 Compatible

Operating System

macOS 12+, Ubuntu 20.04+, or Windows 10+ (WSL2 recommended).

Runtime Environment

Node.js LTS (v18.x or v20.x) or Python 3.10+ required.

Package Manager

Latest version of npm, yarn, or pnpm globally installed.

Network Connection

Active internet connection for package fetching and sync.

Pro Tip: You can run devbase doctor after installation to automatically verify these requirements.
